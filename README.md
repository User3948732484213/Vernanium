# Vernanium
Vernanium - это Python скрипт, предназначенный для использования шифра Вернама.

Шифр Вернама - математический стойкий шифр, позволяющий шифровать и расшифровывать информацию симметричным ключом, складывая исходную информацию с ключом алгоритмом XOR.

Важные критерии для ключа:

1. Ключ должен быть одинаковым по длине информации.

2. Ключ должен быть сгенерирован физическими, квантовыми процессами (библиотека OpenSSL, связка Termux + OpenSSL, если мобильное устройство на Android)

3. Ключ не должен использоваться дважды. При каждом следующим сообщением или куском информации генерируется другой ключ.

Запуск:

1. Скачать "vernanium_bin_xor.py".

2. Открыть этот файл через редактор кода.

3. Интерфейс в консоли.

4. Готово!
